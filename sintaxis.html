Sintaxis Básica de PHP

A continuación se muestra una tabla con ejemplos de sintaxis básica de PHP y una breve descripción de lo que hace cada fragmento de código:

Código PHP Descripción
<?php echo "¡Hola, mundo!"; ?> Imprime "¡Hola, mundo!" en la página web.
<?php $nombre = "Juan"; echo "¡Hola, $nombre!"; ?> Declara una variable llamada $nombre y la inicializa con el valor "Juan", luego imprime "¡Hola, Juan!" en la página web.
<?php $numero = 10; if ($numero > 0) { echo "El número es positivo."; } else { echo "El número es negativo o cero."; } ?> Declara una variable llamada $numero y la inicializa con el valor 10, luego verifica si el número es mayor que cero y muestra un mensaje correspondiente.
<?php for ($i = 1; $i <= 5; $i++) { echo "Número: $i <br>"; } ?> Utiliza un bucle for para imprimir los números del 1 al 5 en la página web.
